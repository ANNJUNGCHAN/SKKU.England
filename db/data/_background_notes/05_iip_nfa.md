# IIP·순대외자산 정의 발췌 (background-search 5회차)

본 문서는 `db/CHECKLIST.md` §0.2 다섯 번째 항목의 산출물로, `background/BoP.pptx`와 `background/BoP.pdf`를 병행 참조해 IIP(국제투자대조표)와 순대외자산(NFA, Net External Assets) 개념을 발췌한 결과이다.

## 요약

- 강의 자료는 IIP를 "특정 시점의 대외투자 잔액(stock)"으로, BoP를 "일정 기간 거래 흐름(flow)"으로 명확히 대비시킨다(슬라이드 25). IIP의 증감은 **거래요인 + 비거래요인**으로 분해되며, 거래요인은 BoP의 "투자수지(=금융계정) + 준비자산증감" 합계와 일치한다고 명시(슬라이드 25).
- **순대외자산(NFA)** 은 두 곳에서 등장: 슬라이드 11("금융수지(준비자산 제외) 흑자 = 대외순자산 증가"), 슬라이드 24("(X−IM) = ΔNFA"). 양자는 BoP 항등식(CA = FA + Reserve)을 매개로 일관된다.
- 강의 자료는 IIP의 **자산·부채 세부 분류표**를 본문 텍스트로는 담지 않으며, 슬라이드 25–26의 그림(`Picture 2`)에만 시각적으로 제시되어 있다(텍스트 추출 불가, §0.2 신규 항목으로 이월).
- 영국 BoP/IIP 적용에서는 만성적 순채무국 특성과 ΔIIP의 가격·환율 재평가 비중이 거래 흐름보다 큰 점이 핵심이며, 이는 강의 자료에서는 일반론으로만 언급되어 ONS 데이터(`db/REPORT.md` 3.3절)로 보완해야 한다.

**슬라이드 11과 24의 일관성 점검**: 두 진술은 항등식 CA = FA(narrow) + Reserve(슬라이드 14)를 거치면 동일하다. 슬라이드 24의 NFA는 통화당국 대차대조표의 **순외화자산**(중앙은행 보유) 관점이고, 슬라이드 11의 NFA는 **국가 전체의 비준비 대외순자산** 관점이라 엄밀히는 적용 범위가 다르지만, 강의 자료는 두 NFA를 동일 기호로 사용해 직관적 동등성을 강조한다. 따라서 **두 NFA 정의는 정성적으로 일관**되나 정량 비교 시에는 (a) 중앙은행 NFA(=준비자산 ± 일부) 와 (b) 국가 전체 Net IIP를 분리해야 한다.

## BoP(flow) ↔ IIP(stock) 항목 매핑 표

| BoP 금융계정 항목(slide 6) | IIP에서의 대응(잔액) | 영국 ONS 표 위치 |
|---|---|---|
| 직접투자(direct investment) | 직접투자 자산 / 직접투자 부채 | `Table_D1_3` 등 (REPORT §3.3) |
| 증권투자(portfolio investment) — 주식·채권 구분 | 증권투자 자산 / 부채 (지분·부채성증권) | `Table_D4_6` |
| 파생금융상품(derivatives) | 파생금융상품 순포지션(자산−부채) | `Table_D7_9` |
| 기타투자(other investment) — 대출·차입·무역신용·예금 | 기타투자 자산 / 부채 | `Table_D7_9` |
| 준비자산증감(changes in reserve assets) | 준비자산 잔액 | `Table_K` (Net IIP 포함) |
| (= 합계) 금융수지 | (= 합계) Net IIP = 총자산 − 총부채 | `Table_K` |

핵심 관계식(슬라이드 25 + 14 종합):

```
ΔNet IIP(t) = [FA(t) + ΔReserve(t)]   ← 거래요인 (= BoP의 거래분)
            + [재평가, 환율, 기타조정](t)  ← 비거래요인
```

## 영국 ONS 적용 주의점

1. **만성적 순채무국**: `db/REPORT.md` §3.3에 따르면 영국 2025 Q4 말 순대외부채는 약 200십억 파운드(GDP의 약 7%) 수준 → NFA < 0 (강의 자료 슬라이드 11의 "순차입 증가" 상태에 해당).
2. **재평가 효과 우세**: REPORT는 "분기별 변동은 환율·자산가격 재평가가 거래 흐름보다 큰 영향을 미친다"고 적시 → 강의 자료 슬라이드 25의 **비거래요인** 항이 영국에서는 거래요인보다 정량적으로 크다는 의미. 따라서 BoP 금융수지만으로 ΔNet IIP를 추정하면 큰 오차가 발생.
3. **부호 체계(슬라이드 8)**: 영국 ONS도 BPM6 채택 → IIP 자산↑ = (+), 부채↑ = (+); 한국은행 신매뉴얼과 동일.
4. **(X−IM) = ΔNFA의 한계**: 슬라이드 24 식 (3-4)는 자본계정·이전소득·재평가를 모두 0으로 가정한 단순화. 영국처럼 1차소득 적자가 큰 경제에서는 (X−IM) 대신 **CA 전체**를 써야 하며, 그래도 재평가로 인해 잔차가 큼.
