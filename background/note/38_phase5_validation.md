## §1 헤드라인 수치 강의 BoP 해석 프레임

### 1.1 상품수지 −£65.5bn → 슬라이드 5·22 「Y < A」

- 슬라이드 5: "재화의 수출입, net trade in merchandise goods"
- 슬라이드 22 (NIA): `Y = C + I + G + (EX − IM)` → `Y − A = CA`. 상품수지 큰 적자 = **국내 흡수 A > 소득 Y**, 차액을 외국 수입으로 충당
- 2025 Q4: 사상 최대 적자 −£65.5bn. 흡수 접근법으로 영국은 "Y < A" 구조화
- 슬라이드 21 (영국 ONS는 `Trade in goods` vs `Total trade balance` 분리): 헤드라인 −£65.5bn은 goods only

### 1.2 서비스수지 +£53.3bn → 슬라이드 5·24 「(EX−IM)>0 → ΔNFA↑」

- 슬라이드 5: 운수·여행·로열티 포함
- 슬라이드 24: `(X − IM) = ΔNFA` (식 3-4). 서비스 흑자 → NFA↑ 효과
- 2025 Q4: +£53.3bn. 그러나 상품 적자 −£65.5bn이 더 커서 net trade는 음수
- 영국 만성 순채무국(Net IIP −£199.8bn) — 식 (3-4)에 "재평가·1차소득 0 가정" 무너짐
- EBOPS 12분류 중 ⑦ Financial · ⑧ IP · ⑩ Other business 주축

### 1.3 경상수지 −£18.4bn → 슬라이드 14 항등식 `CA = G + S + PI + SI`

| 항목 | 값 (£bn) |
|---|---:|
| 상품무역 (G) | −65.5 |
| 서비스무역 (S) | +53.3 |
| 1차소득 (PI) | −2.7 |
| 2차소득 (SI) | −3.6 |
| **합계** | **−18.5** |
| ONS 헤드라인 CA | **−18.4** |

→ 반올림 오차 0.1bn 이내로 강의 항등식 그대로 성립

### 1.4 CA/GDP −2.4% → 슬라이드 15 정규화 방식

- 슬라이드 15: 한국 CA/GDP·FA/GDP 군집 막대(2000~2025), 세로축 −1%~+8%
- 2025 Q4 영국 −2.4%. 한국이 통상 양(+) 영역 vs **영국은 음(−) 영역에 구조적으로 머무는 만성 적자국**
- GDP 대비 정규화는 단위(£·$) 차이 흡수해 직접 비교 가능

### 1.5 Net IIP −£199.8bn → 슬라이드 11·24·25 「NFA = 자산 − 부채」 만성 순채무국

- 슬라이드 11: "금융수지(준비자산 제외) 적자 = 순차입 증가"
- 슬라이드 25: BoP↔IIP 거래요인 + 비거래요인 분해
- 2025 Q4: Net IIP −£199.8bn. GDP 대비 약 7% 순부채
- **재평가 우세**: 분기간 변동(2024 Q4 −172.4 → 2025 Q1 −305.7 → 2025 Q4 −199.8)이 거래 흐름만으로 설명 안 됨 — 비거래요인(가격·환율) 영향

## §2 슬라이드 14 항등식 `CA + KA + FA + NEO ≡ 0` 검산 자문

### 2.1 강의 자료 항등식 표현

표적 항등식은 명시 식 형태로 적지 않고 3 슬라이드 결합으로 도출:
- 슬라이드 13 복식부기 사후 항등성
- 슬라이드 14 `CA = FA(broad)` 단순화 항등식("statistical discrepancy = capital account = 0" 가정)
- 슬라이드 6 NEO 정의

### 2.2 BPM6 부호 약속과 ONS 매핑

BPM6 식: `CA + KA − FA + NEO = 0` (FA는 자산 순취득 양수). ONS는 FA를 부호 반전 규약으로 표시 → `CA + KA + (−FA) + NEO = 0`에서 ONS 그대로 합산하면 NEO 일치

### 2.3 2025 Q4 ONS 실측 항등식 검산

| 강의 기호 | ONS 항목 | CDID 후보 | 2025 Q4 (£bn) |
|---|---|---|---:|
| CA | 경상수지 합계 | HBOP | −18.4 |
| KA | 자본수지 | FNVQ | −0.8 |
| FA(broad, ONS 표시) | 직접+증권+파생+기타+준비자산 합 | (HBNT 등) | −9.5 |
| NEO | 순오차·누락 | HHDH | +3.5 |
| **합계** | | | **−25.2** |

**관찰 1**: 헤드라인 부호 합계만으로는 0이 아님 → 시트별 부호 규약 차이 가능성 → Phase 5.1 §3 RDB 조회로 자동 검증 필요

**관찰 2 (정상 변동 범위)**: 영국 NEO 분기 절대값 평균 약 5,872 £m·표본 SD 약 6,840·|NEO|/GDP 평균 0.92%. 본 분기 NEO +3.5bn(약 0.45%)는 정상 범위

**관찰 3 (보조 검산)**: 슬라이드 12·14 "종합수지 − 준비자산증감 = 0". 2025 Q4 준비자산 +0.5bn → 종합수지 +0.5bn 정도여야 하나 실제 합 −29.2bn — NEO·부호 규약 정합으로 설명되어야 함

### 2.4 자문 결론

- **강의 기대 vs 실측**: 강의는 "통계 불일치 = 0" 단순화 가정만 명시 → 영국 실측에 그대로 적용하면 NEO ≠ 0 잔차. 강의 한계이지 데이터 오류 아님
- **본 분기 NEO 정상**: +3.5bn은 24분기 표본 평균·SD 안에 들어감
- **부호 합계 −25.2bn 잔차**: ONS 시트별 부호 규약 차이 + 헤드라인 표 vs 분기 흐름표 NEO 부호 차이 가능성
