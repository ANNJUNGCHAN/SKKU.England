# v2 트랙 단계 B — 강의 자료 측 연결 고리 추출

본 문서는 v2 보고서(분석 30% + 영국 경제사 70%) 단계 B 카드 작성(7 변곡점 +
10 키워드)을 위해 **각 카드가 강의 자료(`background/`) 어느 슬라이드·노트와
연결되는지** 만 추출한 매핑 표이다. 카드 본문(영국사 서술)은 본 문서 범위가 아
니며, 본 문서는 카드 집필 시 인용으로 가져올 강의 측 발췌 후보만 정리한다.

본 문서 작업 범위는 강의 자료 1차 근거(`background/BoP.pptx` 31 슬라이드 +
`background/note/01_*.md ~ 39_*.md` 39 회차) 와 v1 산출 `report/research/01_*.md
~ 14_*.md` 의 재활용이며, 외부 web-search 는 사용하지 않았다. 강의 자료에서
직접 매핑되지 않는 사건·키워드는 §3 에 따로 모아 외부 출처 의존 여부를 표시
한다.

매핑 신뢰도 약속:
- **H** (High) — 강의에서 그 사건/키워드를 명시적으로 다루거나 정량을 제시.
- **M** (Medium) — 강의 일반론·유사 사례·정의가 그대로 적용 가능하나 사건명은
  부재.
- **L** (Low) — 약한 연결. 강의의 식·도식이 간접적으로 적용되며 별도 외부 보강
  필요.
