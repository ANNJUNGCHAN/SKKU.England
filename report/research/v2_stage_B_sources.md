## §4. 검색 실패·접근 차단 메모

1. **ONS Balance of Payments 2008Q4·2009Q1 분기 보도자료 직접 페이지 미확보**
   - 검색에서는 ONS BoP previous releases 인덱스 (https://www.ons.gov.uk/economy/nationalaccounts/balanceofpayments/bulletins/balanceofpayments/previousreleases) 만 확인. 2008Q4 분기보도 (통상 2009-03 발표) 의 직접 URL 은 해당 인덱스에서 수동 탐색 필요.

2. **ONS Economic Review 2017-03·2017-09 (Brexit 언급분) 직접 페이지 미확보**
   - 검색 결과에 명확한 단일 URL 이 잡히지 않음. ONS Economic Review 자체는 https://www.ons.gov.uk/economy/nationalaccounts/uksectoraccounts/compendium/economicreview/ 인덱스에서 월별로 탐색 필요.
   - 대체: B-1-3 행 2·3 에 ONS 2017-09-15 제조업 sterling 분석과 ONS Economic Review 2019-07-18 ERPT 분석이 이미 들어감.
3. **HM Treasury 2022-09-23 Growth Plan 보도자료 — 별도 보도자료 페이지**
   - GOV.UK 의 보도자료 "Chancellor announces new Growth Plan with biggest package of tax cuts in generations" (https://www.gov.uk/government/news/chancellor-announces-new-growth-plan-with-biggest-package-of-tax-cuts-in-generations) 도 1차 출처로 추가 가능. 본 표에는 연설문 페이지 (speeches/the-growth-plan-2022-speech) 를 우선 등재.
4. **BoE 운영 도메인 일부 페이지 직접 fetch 차단** (v1 §6 에서도 보고됨)
   - `wwwtest.bankofengland.co.uk` 로 우회되는 결과가 있어, 보고서 본문 인용 시에는 운영 도메인 `www.bankofengland.co.uk` URL 을 그대로 사용하고, 가능한 경우 페이지 캡처를 부록에 첨부 권장.
